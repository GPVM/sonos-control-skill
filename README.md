## Sonos Control Skill
Control sonos speakers from Mycroft

## Description
Allow control of sonos speakers from Mycroft.
Assumes all speakers are in one group.

## Examples
 - "Sonos play"  Start playing whatever was playing before
 - "Sonos pause"
 - "Sonos skip"
 - "Sonos volume up"
 - "Sonos volume down"
